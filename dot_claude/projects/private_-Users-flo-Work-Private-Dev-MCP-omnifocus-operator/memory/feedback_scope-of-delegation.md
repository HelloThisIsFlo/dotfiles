---
name: Scope of delegation
description: When user asks Claude to self-check "this one" test/item, do exactly that one — not all remaining items
type: feedback
---

When the user delegates a single item ("check yourself just this one"), do ONLY that item and return to the normal flow for the rest.

**Why:** Flo values agency. "Just this one" means one. Running through everything autonomously violates the core principle — he decides what gets executed.

**How to apply:** After self-checking the delegated item, present the next item back to the user in the normal interactive flow. Never extrapolate a single delegation into blanket permission.
