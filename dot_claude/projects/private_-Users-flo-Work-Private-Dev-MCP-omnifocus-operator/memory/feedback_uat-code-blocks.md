---
name: UAT commands in code blocks
description: During UAT, put runnable commands in fenced code blocks so user can copy-paste easily
type: feedback
---

When presenting UAT checkpoints with commands for the user to run, put each command in its own fenced code block — not inline backticks.

**Why:** User copies commands to run them manually. Inline backticks require manual selection; code blocks have one-click copy.

**How to apply:** Any time a UAT checkpoint includes a command the user should run, use a fenced code block.
